# Bike Sales Dashboard - Excel Project

## Project Overview
This Excel project analyzes customer demographic and lifestyle data to understand bike purchase behavior. The workbook includes cleaned data, working analysis sheets, pivot tables, and an interactive dashboard designed to support business decision making.

## Objective
To identify customer patterns and sales trends that can help a business understand which customer groups are more likely to purchase bikes.

## Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Workbook Structure
- `bike_buyers`: Original dataset with customer details such as income, age, gender, education, occupation, commute distance, region, and bike purchase status.
- `working sheet`: Cleaned and prepared data used for analysis.
- `pivot table`: Pivot table analysis supporting dashboard visuals.
- `dashboard`: Interactive Bike Sales Dashboard with charts and slicers.

## Key Analysis Areas
- Bike purchase behavior by gender, marital status, and region
- Income comparison between bike buyers and non-buyers
- Customer age group analysis
- Commute distance impact on bike purchase decisions
- Occupation and education-based purchasing patterns

## Dashboard Features
- Interactive slicers for quick filtering
- Pivot charts for visual analysis
- KPI style summary views
- Clean dashboard layout for business presentation

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Pivot Table Analysis
- Data Visualization
- Dashboard Development
